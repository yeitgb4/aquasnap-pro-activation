## AquaSnap 到底能做什么

这软件的核心逻辑很简单：让你的窗口"听话"。

**窗口边缘停靠（AquaSnap）**
把窗口拖到屏幕边缘或角落，它会自动撑满半屏或四分之一屏。这个功能 Windows 10 本身也有，但 AquaSnap 的版本自定义选项多得多，在 4K 大屏或多显示器环境下特别好用。

**窗口磁吸对齐（AquaDock）**
给窗口加了"磁力"，让你手动拖动时自动吸附对齐，再也不用逐像素去扣位置。那种把窗口摆得歪歪扭扭的强迫症噩梦，直接消失。

**双击边缘拉伸（AquaStretch）**
双击窗口边缘，它就往那个方向撑满可用空间。想让某个窗口在垂直方向铺满全屏但保留水平宽度？一下搞定。

**晃动置顶透明（AquaShake）**
这个设计有点好玩——把一个窗口使劲摇一摇，它会变成半透明并置顶显示。适合那种需要一直盯着某个小浮窗（比如实时计算器、翻译工具）同时操作其他软件的场景。再摇一下，恢复正常。

**同时移动多个窗口（Multi-move）**
按住 Ctrl 移动一个窗口，旁边并排的相邻窗口也会跟着移动，整个布局平移，不会打乱相对位置。这个功能免费版没有，是专业版独占的。

**支持多显示器**
这也是专业版才有的功能。如果你用双屏或者三屏工作，这一条直接是刚需，免费版不支持跨显示器管理。
